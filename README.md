# memesounds

A Python package for playing meme sounds.

## Quick start

```python
import memesounds

memesounds.fart()
memesounds.play_sound("six_seven")
memesounds.play_sound("67")
memesounds.random_sound()
THERE ARE OTHER 20 SOUNDS TRY IT OUT..

## Command line

From the project folder:

```powershell
python -m memesounds --list
python -m memesounds fart
python -m memesounds 67
python -m memesounds --random
```

After installing the package, you can also use:

```powershell
memesounds --list
memesounds fart
```

## Available sounds

- `baby`
- `bone_crack`
- `fart`
- `oppa`
- `siren`
- `six_seven`
- `wow`
